Music Widget Default

Music Widget for Spotify

Configuration

OBS Studio

1. Make a browser source in OBS Studio with the settings: Width = 1920, Height = 1080.
2. Put the URL of the widget in the Browser Source, find URL here.

Widget

Put this URL into OBS: https://ik1497.github.io/Extensions/Music/Widget/

URL Parameters

  • URL Parameters An explanation, on how to use these parameters in your browser source

Font

Name Description
font-family The Font Family Case Sensitive Default: Montserrat.
color-primary The primary text color, used for song title.
color-accent The accent text color, used for song artist and the song duration.
alt-background-color The alternative background color, used for album cover background.

Misc

Name Description
ws The websocket server of streamer.bot default: ws://localhost:8080/.

Transition

Name Description
size-delay The delay between transitioning from small to big and vice versa Default: 8000ms
transition-time The time it takes to transition from the small to the big size and vice versa Default: 1s
transition-timing-function The transition timing function Default: ease

Background

Name Description
background-opacity The background opacity of the widget.
background-hue The background hue of the widget Choose between: 0-360, recommended: 0, 30, 230, 240 or 345.
background-saturation The background saturation of the widget Choose between: 0-100,
used for how much hue gets used.
background-lightness1 The background background lightness top, Choose between 0-100, 0: light, 100: dark
Used for how light/dark the hue is.
background-lightness2 The background background lightness bottom, Choose between 0-100, 0: light, 100: dark
Used for how light/dark the hue is.

Sizing

Name Description
canvas-spacing The spacing on the bottom and left of the widget Default: 1em.
height The height of the entire widget.
width The width of the entire widget.
border-radius The rounded corners of the widget.
height-small The height of the entire widget when it's is in it's small state.
border-radius-small The rounded corners when the widget is in it's small state.

Visibility

Name Description
progress-bar-hidden Hides the progress bar.
