Music Widget Minimal

Music Widget for Spotify

Configuration

Streamer.bot

1. Make sure that the Server/Clients --> Websocket Server is enabled.
2. You don't have to import any action and/or connect something to events, this the only thing!

OBS Studio

1. Make a browser source in OBS Studio with the settings: Width = 1920, Height = 1080.
2. Put the URL of the widget in the Browser Source, find URL here.

Widget

Put this URL into OBS: https://ik1497.github.io/Extensions/Music/Minimal/

URL Parameters

  • URL Parameters An explanation, on how to use these parameters in your browser source

Font

Name Description
font-family The Font Family Case Sensitive Default: Uni Sans CAPS.
font-style The font style, default: normal.
font-size The font size, default: 1em.

Misc

Name Description
ws The websocket server of streamer.bot default: ws://localhost:8080/.
border-radius-album-cover The border radius on the album cover default: 1em.
border-radius-texts-background The border radius of the background from the texts default: 1.5em.
canvas-spacing The spacing on the bottom and left from the widget default: 1em.

Colors

Name Description
color-primary The primary text color, used for song title default: hsl(0, 0%, 100%).
color-accent The accent text color, used for song artist default: hsla(0, 0%, 100%, 0.7).

Background

Name Description
background The background behind the texts default: hsl(0, 0%, 0%, 0.7).
alt-background The background of the album cover default: hsl(240, 5%, 11%).

Make it your own with URL Parameters! This example image below uses:
https://ik1497.github.io/Extensions/Music/Minimal/?font-style=italic&border-radius-album-cover=0&border-radius-texts-background=0&background=hsl(0, 0%, 100%, 0.7)&color-primary=hsl(0, 0%, 0%)&color-accent=hsla(0, 0%, 0%, 0.7)
