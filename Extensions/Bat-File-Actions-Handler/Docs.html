Prerequisites

- Node.js: Node.js is needed to send out the action via the command line to Streamer.bot. Install: https://nodejs.org/en/ you can choose either versions. To verify it's installed correctly go the your command line (windows + r --> cmd) and type node -v and if it says v..... you have installed it correctly.

- HTTP Server: in Streamer.bot the HTTP Server needs to be enabled, you can leave it at the default settings, but if you choose to change it go to the action.cjs file and change the settings at the top of the file.

Installation

- In the .zip file attached there is a folder, this by default contains two files: action.cjs, actions.bat you don't have to touch the action.cjs, although you need to change the action.bat file, to do this you can right-click and click on Edit, here you're gonna see: node action.cjs ActionName change the ActionName to your action name and your done.

For actions with spaces you need to use {space}

Action.bat Examples

node action.cjs Start Poll
node action.cjs Pause Channel Points
