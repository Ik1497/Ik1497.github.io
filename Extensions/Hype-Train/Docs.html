Hype Train Widget

Show your Hype Train on screen!

Configuration

Streamer.bot

1. Make sure that the Server/Clients --> Websocket Server is enabled.
2. You don't have to import any action and/or connect something to events, this the only thing!

OBS Studio

1. Make a browser source in OBS Studio with the settings: Width = 1920, Height = 1080.
2. Put the URL of the widget in the Browser Source, find URL here.

Widget

Put this URL into OBS: https://ik1497.github.io/Extensions/Hype-Train/

URL Parameters

  • URL Parameters An explanation, on how to use these parameters in your browser source

Font

Name Description
font-family The font family, default: Uni Sans CAPS.
font-weight The font weight, default: 900.
font-style The font style, default: normal.
font-size The font size, default: 1em.
font-color The font color: default: #fff

Misc

Name Description
ws The websocket server of streamer.bot default: ws://localhost:8080/.
background The background of the widget default: hsl(0, 0%, 0%, 70%).
background-progress-bar The color of the progress bar default: hsl(277, 100%, 72%).
border-radius The border radius around the widget default: .5em.
