URL Edits

An explanation, on how to use the custom properties in your browser source

Overview

In the url after the website info you can put extra info. One of these is the ? and the &.

End result: https://example.com?key=value&key=value.

key

The key is the thing before the value, this is so the website knows what setting you want to change.

value

The value is the thing after the key, this is so the website knows what value you want that setting to be.

=

This is always between the key and value.

?

The questionmark is whats directly after the URL, following the key, value, and then optionally an &.

&

The ampersand is for after the first key and value, you can't do a questionmark here.
