URL Parameters

  • URL Parameters An explanation, on how to use these parameters in your browser source

Font

Name Description
font-family The font family, default: Uni Sans CAPS.
font-weight The font weight, default: 900.
font-style The font style, default: normal.
font-size The font size, default: 1.5em.
font-color The font and the svg color: default: #181818

Background

Name Description
background Change the background color, allows CSS gradients.
Doesn't allow hex codes, use something like HSL, RGB, etc. (this will remove day/night background and only use this one).
Default Day: linear-gradient(hsl(194, 97%, 44%), hsl(219, 94%, 41%)).
Default Night: linear-gradient(hsl(224, 14%, 15%), hsl(255, 20%, 4%)).

Misc

Name Description
lang The language of the widget default: en.
This is auto translated, if you spot wrong translations, send me a DM on Discord.
Choose between: ar, cs, da, de, el, en, es, fi, fr, it, ja, nl, no, pl, pt, ro, ru, sv, uk, zh.
force-time If you want the widget to have the same look in night and day no default.
Choose between day and night.
Note: the background and icons change with this.
animation-off Turns the animated icons off.
