How to transfer to OBS Websocket version 5 without breaking all my OBS sub-actions? 

This won't be needed after v0.1.14


1: Disconnect your OBS Connection. | DON'T DELETE IT.
2: Edit the OBS Connection to version `v5.x` and port `4455`
and make sure you have OBS Websocket version 5 installed.
2.5: if authentication is enabled in OBS websocket
    make sure the passwords still match
3: And your done! Your sub-actions should behave fine.
4: If you used OBS raw then check this below ⬇⬇⬇⬇

https://wiki.streamer.bot/en/Broadcasters/OBS/Requests
https://obs-raw.streamer.bot/

If you did remove it do this:
1: Restore a backup, guide can be found below ⬇⬇⬇⬇

https://wiki.streamer.bot/en/Backup
